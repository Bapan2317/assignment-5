Question-1: What are the different ways to select an element in the DOM?

To select an element in the DOM, you can use methods like getElementByID, getElementsByClassName, querySelector, querySelectorAll, document.body etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML is a tag that selects the text and shows the pure HTML structure, innerText is a tag that selects the text and shows the value of the text.

Question-3: What is event delegation in the DOM?

A technique where you attach a single event listener to a parent element, allowing it to handle events triggered on any of its child elements.

Question-4: What is event bubbling in the DOM?

The mechanism where an event triggered on a child element propagates upwards through its parent elements in the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element, the document.createElement method is used, allowing the creation of new HTML elements such as div, p, or button.
